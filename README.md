# Rangeway Concept Site

Premium EV hospitality brand concept website.

**Live:** [concept.rangewayev.com](https://concept.rangewayev.com)

## Overview

This Jekyll site showcases Rangeway's vision for hospitality-driven EV charging. Built with the brand's concept color palette:

- **Charcoal** #2d2d2d
- **Warm** #f4a855
- **Cream** #f5f1eb
- **Sage** #4a5d52

## Development

```bash
bundle install
bundle exec jekyll serve
```

## Deployment

Pushes to `main` automatically deploy to GitHub Pages.
